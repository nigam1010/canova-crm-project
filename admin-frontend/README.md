# CanovaCRM Admin Frontend

## Setup

```bash
npm install
npm run dev
```

Admin panel will open at `http://localhost:3000`

## Default Login
- Email: admin@canovacrm.com
- Password: admin123

## Features
- Dashboard with KPIs and analytics
- Employee management (CRUD, pagination, bulk delete)
- Lead management (CSV upload, manual add)
- Settings (profile update)

## Tech Stack
- React 18
- React Router
- Vite
- Vanilla CSS
